# api

### Запуск миграций:

`
npm run migrate
`
### Откат последней миграции:

`
npm run migrate:undo
`

### Откат всех миграций:

`
npm run migrate:undo:all
`

### Запуск сеятеля данных:

`
npm run seed
`

### Откат последнего:

`
npm run seed:undo
`

### Откат всех:

`
npm run seed:undo:all
`
